let / const variable declarations

1. when let & const are declared on a global level, the variables are not stored as properties of
gloabl window object. But, they still act as global objects.
2. let & const are function and block scope variable declarations.
3. Parts of const can be manipulated when used as an object.
4. When let & const are run in console b4 being declared,
the value of variable becomes undefined. While it works for "var".
5. let & var variables can be redeclared while consts' cannot.
6. Let variable without a value will be undefined while const variables syntax error is given.
